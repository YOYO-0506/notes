# Git

## 初始化仓库
```shell
git init 
```

## 添加代码变更列表到暂存列表
```shell
git add .
```

## 确认存储
```shell
git commit -m "描述"
```

## 查看分支
```shell
git branch
```

## 创建分支
```shell
git branch <branchname>
```

## 切换分支
```shell
git checkout <branchname>
```

## 删除分支
```shell
git branch -d <branchname>
```

## 合并分支
```shell
git merge
```
